# RAG Chatbot: Retrieval-Augmented Generation Chat Assistant

This project implements a memory based Retrieval-Augmented Generation chatbot that combines the power of Large Language Models (LLMs) with information retrieval techniques to provide accurate, context-aware answers based on a custom knowledge base.


## Tech Stack

- Python
- LangChain
- Pinecone
- Hugging Face Embedding
- Gemma model

## Features

- Scraping data from different websites
- Chunking and embedding of document content
- Vector search for retrieving relevant context
- LLM responds based on retrieved context
- Memory & session handling
